## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/my-portfolio.git
```

Go to the project directory

```bash
  cd ..my-portfolio
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
